MODELO PEDAGÓGICO

Nuestro Modelo Pedagógico

El colegio el placer de aprender concibe el aprendizaje como un proceso mediante el
cual el ser humano modifica y adquiere aquellas habilidades, destrezas,
conocimientos, conductas o valores necesarios para su desarrollo personal, y entiende
que este proceso valida varios escenarios. Por eso potencializamos las inteligencias
múltiples

"Todo lo que vale la pena enseñar se puede presentar de muchas maneras diferentes.
Estas formas múltiples pueden hacer uso de nuestras inteligencias múltiples".
